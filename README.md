# Ping Paddle

A classic paddle-and-ball game inspired by Pong (1972), built with Godot 4.5 and GDScript.

## Features

- 2-player local (W/S vs O/L)
- AI opponent with adjustable difficulty
- Ball speed challenge slider
- Paddle size slider
- First to 7 wins
- Ball trail effect

## Controls

| Key | Action |
|-----|--------|
| W / S | Player 1 up / down |
| O / L | Player 2 up / down |
| P | Pause |
| C | Continue |
| R | Reset match |

## Running

Open the project in Godot 4.5+ and hit Play, or run an exported build.
