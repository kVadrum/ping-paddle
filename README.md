# Ping Paddle

A classic paddle-and-ball game inspired by Pong (1972), built with Godot 4.5 and GDScript. Zero dependencies, pure procedural rendering.

## Features

- 2-player local (W/S vs O/L) or vs AI opponent
- Adjustable ball speed, paddle size, and AI difficulty
- Dark/light theme toggle
- Window management (fullscreen, windowed, minimize)
- Ball trail effect
- Title screen

## Controls

| Key | Action |
|-----|--------|
| W / S | Player 1 up / down |
| O / L | Player 2 up / down |
| P | Pause |
| G | Go (unpause) |
| R | Reset match |
| T | Toggle dark/light theme |
| Esc | Quit |

First to 7 wins. Use the bottom bar sliders to adjust speed, paddle size, and AI difficulty.

## Running

Requires [Godot 4.5+](https://godotengine.org/download).

```bash
godot --path .
```

Or open the project in the Godot editor and press Play.
